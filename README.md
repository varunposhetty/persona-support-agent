# Persona Adaptive Support Agent

A support chatbot that adapts responses based on user personas.

## Features

- Persona detection
- RAG-based retrieval using FAISS
- Gemini-powered responses
- Escalation detection
- Streamlit UI

## Personas

- Technical Expert
- Frustrated User
- Business Executive

## Tech Stack

- Python
- LangChain
- FAISS
- Sentence Transformers
- Google Gemini
- Streamlit

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
